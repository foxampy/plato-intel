# 🎯 ВОРОНКА ПРОДАЖ

## Этапы воронки:

```
1. ПРИВЛЕЧЕНИЕ (Awareness)
   │
   ├→ Приветственное сообщение с УТП
   ├→ Демонстрация каталога (95 товаров)
   └→ Акция/спецпредложение на видном месте
   │
   ↓
2. ЗАИНТЕРЕСОВАННОСТЬ (Interest)
   │
   ├→ Умный поиск с автодополнением
   ├→ Фильтры по категории, цене, параметрам
   ├→ Карточка товара с фото и характеристиками
   └→ ИИ-консультант отвечает на вопросы
   │
   ↓
3. ЖЕЛАНИЕ (Desire)
   │
   ├→ Сопутствующие товары (Cross-sell)
   ├→ «С этим покупают» (Up-sell)
   ├→ Отзывы и кейсы применения
   └→ Ограничение по времени/количеству
   │
   ↓
4. ДЕЙСТВИЕ (Action)
   │
   ├→ Быстрое добавление в корзину
   ├→ Оформление в 3 шага
   ├→ Промокод на первую покупку
   └→ Мгновенное подтверждение
   │
   ↓
5. УДЕРЖАНИЕ (Retention)
   │
   ├→ Статус заказа в реальном времени
   ├→ Персональные рекомендации
   ├→ Программа лояльности
   └→ Напоминания о повторной покупке
```

# 💾 БАЗА ДАННЫХ

## Модели (SQLAlchemy):

```python
# users - пользователи
class User(Base):
    user_id = Column(BigInteger, primary_key=True)
    username = Column(String)
    first_name = Column(String)
    last_name = Column(String)
    phone = Column(String)
    email = Column(String)
    company = Column(String)
    discount_level = Column(Integer, default=0)  # 0-10%
    created_at = Column(DateTime, default=func.now())
    last_seen = Column(DateTime)

# conversations - диалоги с ИИ
class Conversation(Base):
    conversation_id = Column(BigInteger, primary_key=True)
    user_id = Column(BigInteger, ForeignKey('users.user_id'))
    messages = Column(JSON)  # История сообщений
    context = Column(JSON)   # Контекст диалога
    created_at = Column(DateTime, default=func.now())

# orders - заказы
class Order(Base):
    order_id = Column(BigInteger, primary_key=True)
    user_id = Column(BigInteger, ForeignKey('users.user_id'))
    order_number = Column(String, unique=True)
    status = Column(Enum(OrderStatus))
    items = Column(JSON)  # Состав заказа
    total_amount = Column(Numeric)
    discount = Column(Numeric, default=0)
    delivery_address = Column(Text)
    created_at = Column(DateTime, default=func.now())

# cart - корзина
class CartItem(Base):
    cart_id = Column(BigInteger, primary_key=True)
    user_id = Column(BigInteger, ForeignKey('users.user_id'))
    product_id = Column(String)
    quantity = Column(Integer, default=1)
    added_at = Column(DateTime, default=func.now())
```
