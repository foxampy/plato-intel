### 3. ✅ Telegram-бот с ИИ-консультантом

**Технологический стек:**
- Python 3.11+
- aiogram 3.x (асинхронный фреймворк)
- Qwen API (ИИ-консультант)
- PostgreSQL 15 (база данных)
- Redis 7 (кэширование)
- Docker (контейнеризация)

**Ключевой функционал:**
- 🛒 Каталог товаров (95 позиций)
- 🔍 Умный поиск
- 📋 Корзина заявок
- 💬 ИИ-консультант (Qwen API)
- 📦 Оформление заказов
- 📊 Админ-панель
- 🎯 Воронка продаж
- 🔄 Cross-sell и Up-sell

### 4. ✅ ИИ-консультант «АДМИНИСТРАТОР»

**Возможности ИИ:**
- Подбор товаров по параметрам
- Ответы на технические вопросы
- Cross-sell предложения
- Up-sell рекомендации
- Обработка возражений
- Консультации по условиям доставки и оплаты

**Сценарии работы:**
1. **Подбор товара** — клиент описывает задачу, ИИ предлагает варианты
2. **Cross-sell** — при добавлении в корзину ИИ предлагает сопутствующие товары
3. **Up-sell** — ИИ предлагает более дорогую альтернативу с обоснованием выгоды
4. **Работа с возражениями** — ИИ обрабатывает возражения типа "дорого", "долго", "сомневаюсь"

### 5. ✅ Воронка продаж

**5 этапов воронки:**

```
1. ПРИВЛЕЧЕНИЕ → Приветствие с УТП, демонстрация каталога
   ↓
2. ЗАИНТЕРЕСОВАННОСТЬ → Умный поиск, фильтры, карточка товара
   ↓
3. ЖЕЛАНИЕ → Сопутствующие товары, отзывы, ограничения
   ↓
4. ДЕЙСТВИЕ → Быстрое оформление, промокод
   ↓
5. УДЕРЖАНИЕ → Статус заказа, рекомендации, программа лояльности
```

### 6. ✅ CRM-система для диалогов

**Таблицы базы данных:**
- `users` — пользователи (ID, имя, телефон, скидка)
- `conversations` — диалоги с ИИ (история, контекст, статус)
- `orders` — заказы (состав, сумма, статус)
- `cart_items` — корзина (товары, количество)
- `products` — товары (кэш из JSON)
- `cross_sells` — сопутствующие товары

**Статусы диалогов:**
- `new` — новый диалог
- `active` — оператор работает
- `waiting` — ожидает ответа клиента
- `closed` — завершён

| Команда | Описание |
|---------|----------|
| `/start` | Запустить бота, главное меню |

| `/profile` | Профиль пользователя |
| `/support` | Поддержка (ИИ-консультант, оператор) |
| `/help` | Помощь, список команд |
| `/admin` | Админ-панель (только для ADMIN_IDS) |

## Сценарии:

### 1. Покупка товара
```
Клиент: /catalog → Выбирает категорию → Выбирает товар →
"В корзину" → Оформить → Вводит адрес → Оплата →
✅ Заказ создан!
```

### 2. Консультация с ИИ
```
Клиент: /support → "Задать вопрос ИИ" →
"Какой тормоз подойдёт для крана 5 тонн?" →
ИИ: "Рекомендую ТКГ-200..." →
Клиент: "Добавить в корзину" → ...
```

### 3. Cross-sell
```
Клиент добавил ТКГ-200 в корзину →
ИИ: "С этим товаром покупают:
• Тормозные колодки — 4 500 ₽
```

## 2. Заполнение .env
```env

DASHSCOPE_API_KEY=sk-ваш_ключ
DATABASE_URL=postgresql+asyncpg://...
ADMIN_IDS=ваш_id
```

## 3. Запуск
```bash
# База данных
docker-compose up -d db redis

# Миграции
python -m database.migrate

# Бот
python -m bot.main
```
