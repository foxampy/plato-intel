## РАЗДЕЛ 2: РАЗРАБОТКА УНИКАЛЬНОГО СТИЛЯ

### 2.1 Концепция: «Тёплый индустриальный неоморфизм»

**Суть:** Современный тактильный дизайн, понятный инженерам 40-50 лет, с узнаваемыми элементами советской инженерии.

**Почему это работает для ЦА:**
- Газоразрядные индикаторы (ГРИ) — вызывают ностальгию без слабости, ассоциируются с надёжностью
- Галетные переключатели — тактильный контроль, привычный промышленным работникам
- Видимая чертёжная сетка — структура, точность, инженерная культура
- Тёплые оранжевые тона — контраст холодным корпоративным сайтам конкурентов

### 2.2 Три направления в единой системе

| Направление | Элементы | Применение |
|-------------|----------|------------|
| **Неоморфизм** | Мягкие тени, выпуклые панели, металлические текстуры | Основной интерфейс, кнопки, карточки товаров |
| **Конструктивизм** | Видимая модульная сетка, геометрия, массивная типографика | Компоновка страниц, акцентные блоки |
| **Ретрофутуризм** | ГРИ-индикаторы, галетные переключатели, приборные панели | Hero-секция, ключевые цифры, интерактивные элементы |

### 2.3 Разработанные компоненты

| Компонент | Описание | Функция |
|-----------|----------|---------|
| **ГРИ-счётчики** | Анимированные газоразрядные цифры с эффектом мерцания | Отображение 20+ лет, 12 мес гарантии, 340+ проектов |
| **Галетный переключатель** | Круглая «крутилка» с фиксированными положениями | Выбор режима поиска: по производителю / назначению / артикулу |
| **Компоновочная схема** | Видимая модульная сетка 12 колонок | Структура, предсказуемость, инженерная логика |
| **Неоморфные карточки** | Выпуклые панели с мягкими тенями | Товары, преимущества, контакты |

## РАЗДЕЛ 3: ВЫВОДЫ И ПРИЧИНЫ РЕШЕНИЙ

### 3.1 Почему текущий подход не работает

| Проблема | Последствие | Наше решение |
|----------|-------------|--------------|
| Шаблонный дизайн | Нет дифференциации, сайт не запоминается | Уникальный стиль, узнаваемость через 3 секунды |
| Отсутствие автоматизации | Менеджеры тратят 60% времени на рутину | ИИ-чат, CRM, автозаявки — экономия 15-20 часов/неделю |
| Нет сквозной аналитики | Не понятно, что работает | Интеграция источник → заявка → сделка → выручка |
| Разрыв сайта и продаж | Потеря 30-40% лидов | Единая система: сайт + чат + бот + CRM |

### 3.2 Почему выбрана автономность + ИИ

| Фактор | Обоснование |
|--------|-------------|
| **ЦА не любит звонить** | Инженеры 40+ предпочитают получить информацию самостоятельно |
| **Рабочее время ограничено** | Заявки вечером и выходные — ИИ обрабатывает 24/7 |
| **Сложные технические вопросы** | ИИ обучен на документации, даёт точные ответы быстрее человека |
| **Масштабируемость** | Рост трафика не требует пропорционального роста штата |

### 3.3 Почему тёплый неоморфизм + конструктивизм

| Психология ЦА | Дизайн-решение |
|---------------|----------------|
| Ностальгия по «вещам на века» | Ретро-элементы, ГРИ, металл |
| Потребность в структуре и порядке | Видимая сетка, модульность |
| Недоверие к «маркетинговой блестяшке» | Матовые поверхности, функциональность |
| Уважение к инженерной культуре | Чертёжная точность, ГОСТ-овская логика |
