## 🔧 РЕШЕНИЕ

### План обновления базы товаров:

#### Этап 1: Парсинг сайта (2-3 часа)

```bash
cd f:\Work\platointel\telegram-bot

# Установка зависимостей
pip install requests beautifulsoup4 fake-useragent lxml

# Запуск парсера
python scripts/scraper.py
```

**Результат:**
- Файл `data/scraped/products-full.json` с ~372 товарами
- Файлы по категориям `products-{category}.json`

#### Этап 2: Объединение данных (30 мин)

```bash
python scripts/merge_products.py
```

**Результат:**
- Файл `data/merged/products-merged.json`
- Удаление дубликатов
- Полная база: ~372 товара

#### Этап 3: Обновление бота (30 мин)

```bash
# Копирование обновлённых данных
copy data\merged\products-merged.json data\products.json

# Перезапуск бота
docker-compose restart bot
```
